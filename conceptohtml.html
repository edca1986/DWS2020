HTML

HyperText Markup Lenguage (HTML) está constituido de elementos para que el navegador los interprete y los visualice de acuerdo con el objetivo trazado, el lenguaje HTML nos sirve para describir la estructura básica de una página y organizar la forma en que se mostrará su interfaz gráfica, además permite incluir enlaces hacia otras páginas o documentos.
