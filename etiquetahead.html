HEAD

Etiqueta que define la cabecera de un documento HTML, incluyendo su título, enlaces a scripts y hojas de estilo.

TITLE:Define el titulo de la barra de herramientas del navegador.

META:Los metadatos pueden ser utilizados por los navegadores (cómo mostrar contenido o recarga de la página), motores de búsqueda (palabras clave), u otros servicios web.

STYLE:Se utiliza para utilizar código css dentro del html

SCRIPT:Se utiliza para definir un script del lado del cliente(JavaScript).
